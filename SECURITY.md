# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Report a vulnerability via an [issue](https://github.com/zeitport/eslint-plugin-json-es/issues) or a direct message to @cschuller.
